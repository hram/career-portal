# Шаг 8 — ИИ-агент: проверка базы знаний

## Контекст
Шаги 1-7 выполнены. Данные есть, роли настроены.  
Первый сценарий работы агента — проверить базу и задать вопросы.

---

## Задача
Страница где агент читает всю базу знаний и выдаёт:
1. Чего не хватает
2. Что нужно уточнить
3. Что выглядит слабо

---

## Service: `app/services/ai_agent.py`

### Функция `analyze_profile(profile_id) -> dict`

Собирает всю базу знаний в один большой контекст:
- Профиль и summary
- Все компании с проектами
- Навыки, образование, заметки

Отправляет в Claude API со следующим системным промптом:

```
Ты — персональный HR-агент и карьерный консультант.
Твоя задача: проанализировать базу знаний кандидата и помочь её улучшить.

Ты не пишешь резюме прямо сейчас. Ты проверяешь данные.

Верни ответ в JSON:
{
  "overall_score": 75,          // 0-100, насколько полна база
  "missing_critical": [],       // чего критически не хватает
  "questions": [],              // конкретные вопросы которые нужно задать
  "weak_points": [],            // что выглядит слабо или расплывчато
  "strengths": [],              // что выглядит сильно
  "recommendations": []         // что сделать прямо сейчас
}

Для questions — конкретные вопросы вида:
"В проекте 'Яндекс.Карты' ты написал 'ускорили загрузку'. На сколько процентов? Это важно для резюме."

Будь конкретным. Не давай общих советов.
```

---

## Router: `app/routers/agent.py`

```
GET  /agent           → страница агента (форма запуска)
POST /agent/analyze   → запустить анализ → вернуть JSON (или Server-Sent Events для стриминга)
```

---

## Template: `app/templates/agent.html`

**До запуска:**
```

```

**Во время анализа:**
```

```

**После анализа — результат:**

Блок "Готовность базы":
```

```

Блок "Нужно добавить":
- список missing_critical красными тегами

Блок "Вопросы агента" (самое важное):
```
❓ В проекте "Яндекс.Карты" ты написал "ускорили загрузку". 
   На сколько процентов? Это важно для резюме.

```

Блок "Сильные стороны":

# Шаг 9 — ИИ-агент: генерация резюме

## Контекст
Шаги 1-8 выполнены. Агент умеет анализировать базу.  
Главная функция портала — генерация резюме под вакансию.

---

## Задача
Страница генерации: выбрать роль + вставить вакансию → получить резюме.

---

## Service: `app/services/ai_agent.py`

### Функция `generate_resume(profile_id, template_id, vacancy_text, company_name) -> str`

Собирает контекст и отправляет в Claude API:

**Системный промпт:**
```
Ты — опытный HR-консультант и копирайтер резюме.
Твоя задача: написать идеальное резюме на русском языке.

Правила:
1. Используй ТОЛЬКО данные из базы знаний кандидата — не выдумывай
2. Адаптируй акценты под роль и конкретную вакансию
3. Встрой ключевые слова из вакансии органично — не механически
4. Метрики и результаты выноси вперёд — это главное
5. Тон соответствует роли: {tone}
6. Длина: 1-1.5 страницы A4 (не больше)
7. Формат: Markdown с чёткой структурой

Структура резюме:
# Имя Фамилия
Контакты в одну строку

## О себе (2-3 предложения, tailored под вакансию)

## Опыт работы
### Компания (период)
**Должность**
- Проект: краткое описание с метриками
- ...

## Навыки
(структурированный список, релевантный вакансии)

## Образование

НЕ добавляй секции которых нет в данных.
НЕ выдумывай достижения.
Если данных мало — напиши что не хватает, не заполняй пустотой.
```

---

## Router: `app/routers/resumes.py`

```
GET  /generate                → форма генерации
POST /generate                → запустить генерацию → сохранить → redirect /resumes/{id}
GET  /resumes                 → список всех резюме
GET  /resumes/{id}            → просмотр резюме
DELETE /resumes/{id}          → удалить
GET  /resumes/{id}/download   → скачать как .md файл
```

---

## Template: `app/templates/generate.html`

**Форма:**
```
Выбери роль:
○ Техлид Android
○ Руководитель отдела разработки
○ Лид бэкенд разработки
○ Фулстек разработчик

Название компании (необязательно):
[________________________]

Ссылка на вакансию (необязательно):
[________________________]

Текст вакансии *:
[большой textarea]
[Вставь полный текст вакансии — требования, обязанности, стек]

[ ✨ Создать резюме ]
```

**Во время генерации:**
```
✨ Агент пишет резюме под эту вакансию...
Подбираю ключевые слова...
Структурирую опыт...
```

---

## Template: `app/templates/resume_view.html`

Двухколоночный layout:

**Левая — готовое резюме:**
Markdown рендерится в HTML (использовать простой JS markdown parser или прислать как HTML от сервера через `markdown` библиотеку Python).

**Правая — метаданные:**
```
Роль: Техлид Android
Компания: Яндекс
Создано: 14 мая 2025

[ 📋 Скопировать текст ]
[ ⬇️ Скачать .md ]
[ 🔄 Перегенерировать ]
[ 🗑 Удалить ]
```

- [ ] Резюме генерируется и сохраняется в БД
